# PennyLane QSVT Demo Notebooks

[PennyLane](https://pennylane.ai/) is an open-source quantum software platform for quantum computing, quantum machine learning, and quantum chemistry. Create meaningful quantum algorithms, from inspiration to implementation. ✨

This repository contains some PennyLane demos in Jupyter notebook format related to [QSVT](https://pennylane.ai/codebook/hamiltonian-simulation/hamiltonian-simulation-qsvt). You can easily launch these notebooks directly in the cloud using qBraid. 🚀

If you like these demos, go to [pennylane.ai](https://pennylane.ai/), to find all things PennyLane, including more demos, blogs, coding challenges, and the one and only [PennyLane Codebook](https://pennylane.ai/codebook). 

> **Note:** The main repository for demos is [PennyLaneAI/demos](https://github.com/PennyLaneAI/demos).

## Getting Started

You can run these notebooks in the cloud or on your local machine.

### Option 1: Launch in the Cloud

🚀 Create a free [qBraid account](https://qbraid.com/).

🚀 Click on the "Launch on qBraid" button below to access the notebooks in your qBraid account. 

[<img src="https://qbraid-static.s3.amazonaws.com/logos/Launch_on_qBraid_white.png" width="150">](https://account.qbraid.com?gitHubUrl=https://github.com/PennyLaneAI/pennylane-demo-notebooks.git)

*(Note: Give it a couple of minutes for qBraid to clone the repository and set up your PennyLane environment.)*

### Option 2: Run Locally

🚀 If you prefer to run the demos locally, simply clone this repository and [install PennyLane](https://pennylane.ai/install).
